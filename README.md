# TintedForge

TintedForge is a library that allows you to easily use Minecraft's color code system when creating a project.

## Color Codes

- Black
- Dark Blue
- Dark Green
- Dark Aqua
- Dark Red
- Dark Purple
- Gold (Orange)
- Gray
- Dark Gray
- Blue
- Green
- Aqua
- Red
- Light Purple
- Yellow
- White

## Formatting Codes

- Obfuscated
- Bold
- Strikethrough
- Underlined
- Italic
- Reset Color/Formatting Code
